Add a user

Administrator can add users manually.

Users list

The list can be filtered on username (use * as wildcard), group or status. It can be ordered by registration date or username, in ascending or descending order.

This screen allows the management of several users at once, using different actions:

  • delete users (requires confirmation)
  • change status
  • associate or dissociate from groups
  • modify display properties
  • modify additional properties

The target is the selected users (by default) or all users as displayed in the filtered list.

Adviser
