Schnellstart

  1. Schaffung eines lokalen Verzeichnis "categorie_test".
  2. Füllen Sie das répetoire "categorie_test" mit Fotos.
  3. Öffnet eine FTP-Verbindung auf der Remote-Galerie.
  4. Download das lokale Verzeichnis "categorie_test" in das Remote-Verzeichnis "Galerien". Geben, alle Rechte (777) zu einem neuen Verzeichnis "Galerien / categorie_test".
  5. Eine Verbindung zu Ihrer Installation von Piwigo als Administrator.
  6. Sich auf Verwaltung, Bilder, Miniaturen. Fragen zu schaffen, die Miniaturansichten fehlen nach der Prüfung von Piwigo.
  7. Sich auf Verwaltung, Kategorien, Synchronisieren . Kontrollkästchen "Verzeichnisse + Dateien" entfernen "simulieren nur "ein und bestätigen Sie die Synchronisierung.
  8. Und das ist es. Sie schaffen die Kategorie "categorie_test" an der Wurzel Ihre Galerie.

  • Allgemeinen eine Kategorie wird durch ein Verzeichnis zu jeder Ebene unter dem Verzeichnis "Galerien" Ihrer Anlage von Piwigo. Jede Kategorie kann beliebig viele Ebenen, die unter - gewünscht. In dem obigen Beispiel, Kategorie-1.1.1.1 ist auf einem Niveau 4 Tiefe.
  • Allgemeinen ein Element wird durch eine Datei. Eine Datei kann ein Element für Piwigo, wenn die Erweiterung des Dateinamens ist aus der Liste der Konfigurationsparameter file_ext (siehe Datei include / config.inc.php). Eine Datei werden kann, wenn ein Bild von der Erweiterung der Konfigurationsparameter picture_ext.
  • Elemente vom Typ Bild müssen eine Miniaturbild (siehe Abschnitt der Miniaturansicht an).
  • Elemente vom Typ Bild können Bilder im Grossformat Partner. Wie für die Datei Hochzeit.jpg im obigen Beispiel. Braucht man keine Vorwahl auf den Namen des Datei.
  • Die Nicht-Bild (Videos, Sounds, Textdateien, die alles, was Sie möchten, ...) werden standardmäßig von einem Symbol für die Erweiterung des Dateinamens. Optional ist eine Miniatur-und Vertreter können (siehe die Datei video.avi im Beispiel).
  • Achtung: den Namen für ein Verzeichnis oder eine Datei darf nur noch aus Buchstaben, Zahlen, "-", "_" oder ".". Kein Platz oder Zeichen mit Akzent.
  • Rates: eine Kategorie kann auch Elemente enthalten, und der Unterkategorien auf einmal. Dennoch ist es empfehlenswert für jeden Kategorie wählen zwischen enthalten oder Unterkategorien.
  • Sobald die Dateien, die Modelle und die Vertreter werden richtig in die Verzeichnisse, sich auf dem Bildschirm Verwaltung, Kategorien, synchronisieren.

Miniaturen

  • Wie bereits erwähnt, jedes Element des Typs Bild muss mit einer Miniatur.
  • Die Miniaturen werden in das Unterverzeichnis "thumbnail" von Vertreter jedes Verzeichnis eine Kategorie. Eine Miniaturansicht ist eine Datei Typ Bild (auch Dateinamenerweiterung), deren Name der Datei Präfix-Einstellungen prefix_thumbnail (siehe include / config.inc.php).
    • Die Miniaturen
  • brauchen nicht einmal die Erweiterung auch, dass ihre Bild (ein Bild mit ". jpg" wie kann eine Erweiterung Miniatur. "GIF" zum Beispiel).
  • Es wird empfohlen, ein externes Programm für die Erstellung von Miniaturen (PhpMyVignettes oder wie ThumbClic finden Sie auf der Website der Übersicht von Piwigo).
  • Sie können auch das Tool zum Erstellen von Miniatur - in Piwigo aber dies wird nicht empfohlen, weil die Gefahr werden dazu verwendet, enttäuschend und unnötig die Ressourcen des Servers (diese werden kann, ein ernstes Problem auf einem Server beziehen).
  • Wenn Sie sich für den Web-Server, um die Miniaturen, müssen Sie die Rechte und Schreibzugriff auf alle Verzeichnisse, die die Kategorien für alle Benutzer (Eigentümer, Gruppe, andere).

Berechtigungen

  • Sie können den Zugriff auf bestimmte Kategorien. Die Kategorien können öffentlich oder privat sein. Die Genehmigungen (für die Gruppen und Benutzer) beherrschbar sind nur für den privaten Gruppen.
  • können Sie eine private veröffentlicht (Verwaltung, Kategorien, Verwalten, Bearbeiten) oder Geschäftsführer die Optionen für Ihren kompletten Baum der Kategorien (Verwaltung, Kategorien, Eigenschaften, Public / Private).
  • Sobald bestimmte Gruppen sind privat, können Sie die Berechtigungen für Benutzer und Gruppen (Verwaltung, Identifikation).

Benutzergruppen

  • Piwigo kann von Gruppen. Sehr praktisch Berechtigungen für die Verwaltung der Gemeinden auf den privaten Gruppen.
  • Sie können Gruppen und Hinzufügen von Benutzern in der Verwaltung, Identifizierung, Gruppen.
  • Ein Benutzer kann mehreren Gruppen angehören. Die Genehmigung wird stärker als das: wenn der Benutzer "Stein" gehört zu den Gruppen "Familie" und "Freunde", und dass nur die Gruppe "Familie" kann sich der Kategorie "Weihnachten 2003", dann "Stein" kann sich dieser Kategorie.

Hinzufügen von Dateien durch die Benutzer

Damit Benutzer das Hinzufügen von Dateien:

  1. erlauben das Hinzufügen von Bildern auf einer beliebigen Kategorie (administ Kategorien, Verwaltung, Bearbeitung oder Verwaltung, Kategorien, Eigenschaften, Hinzufügen von Bildern ).
  2. geben, die Rechte zum Schreiben (für alle Benutzer) auf die Verzeichnisse entsprechen den Kategorien, die zur Hinzufügung.

die hinzugefügten Dateien durch die Benutzer nicht direkt sichtbar auf der Website, sie sind mit einem Administrator. Für dies, muss der Administrator sich im Bereich Verwaltung, Bilder, in Bis zu bestätigen oder ablehnen, die angebotenen Dateien. Es ist dann zu synchronisieren, das System der Datei mit der Datenbank.

Verbindungen zwischen den Elementen und den Kategorien, virtuelle Klassen

  • Piwigo trennt die Kategorien speichern, die die Elemente und die Kategorien, in denen die Elemente angezeigt werden.
  • Standardmäßig werden die Elemente nicht gezeigt werden, dass in ihren Kategorien tatsächlich: solchen, die in ihren Verzeichnissen auf dem Server.
  • So verknüpfen Sie ein Element in einer Kategorie, genügt es, eine Verein auf der Seite zur Bearbeitung des Elements (ein Zusammenhang besteht nach dieser Seite, wenn Sie sich als Administrator), auf der Seite mit Informationen über alle Elemente einer Klasse oder noch mit dem Einkaufswagen.
  • Ausgehend von diesem Prinzip ist es möglich, das Erstellen von Kategorien Abreise: kein Verzeichnis dieser Kategorien entspricht. Sie können Erstellen von virtuellen Gruppen auf Verwaltung, Kategorie, verwalten.
