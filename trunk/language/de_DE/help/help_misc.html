Voreinstellungen für Benutzer festlegen

Nachdem Sie die Galerie erstellt haben, können sie diese nun für ihre Besucher konfigurieren.

  • Die von Ihnen gewünschte Standarddarstellung legen Sie in Administration » Konfiguration » Optionen im Tab Einstellungen für Gäste fest.
  • Das Standardthema legen Sie in Administration » Konfiguration » Themen fest.
  • Die standardmäßig verwendete Sprache legen Sie in Administration » Konfiguration » Sprachen fest.

Damit wird jedem neuen Benutzer diese Einstellung nach erfolgreicher Registrierung vorgegeben.
