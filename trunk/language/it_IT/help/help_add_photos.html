Esistono diversi modi per aggiungere delle foto nella vostra galleria :

  • L'upload diretto dal vostro browser è il modo più semplice e immediato. Questo metodo dipende molto dalle capacità delle vostro browser e del vostro server. Per i grossi volumi o in caso di problemi, dei metodi alternativi vi sono proposti !

Utilizzare l'upload diretto

  • pLoader è un software da installare sul vostro computer. Inserite le vostre foto nella finestra di pLoader, cliquate su "Trasferire verso Piwigo" e pLoader si occupa del resto. Esistono altri software, come WinToPiwigo o il modulo di esportazione Piwigo di Digikam.

Utilizzare pLoader

  • FTP è il modo ideale per aggiungere una collezione importante d'immagini in un minimo di operazioni. Segui la guida per preparare le vostre foto. Questo metodo consente di controllare con precisione l'organizzazione fisica dei file sul server.
