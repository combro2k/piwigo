Ricerca

Cercare delle immagini in tutta la galleria.

Ricerca per parole
Cercare una o più parole in tutti gli attributi che riguardano le immagini mostrate nella galleria. Usare * come carattere Jolly per ricerche parziali.
Ricerca per Autore
Usare * come carattere Jolly per le ricerche parziali.
Ricerca per data
Selezionare una data e/o una data di fine. Lasciare la data vuota se volete fare una ricerca del tipo "prima di". L'anno deve essere inserito nell'ultimo campo come nel formato seguente : 0000 (per esempio 2010)
Ricerca nelle categorie
Scegliere le categorie in cui desiderate eseguire la ricerca. È possibile ricercare in tutte le categorie selezionando la categoria madre e autorizzando la ricerca nelle sottodirectory.
