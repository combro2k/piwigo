Benutzer Liste

Dies ist der Ort, an dem Sie können registrierte Benutzer. Nur Berechtigungen werden in verschiedene mehrere Bildschirme.

Hinzufügen eines neuen Nutzers

Der Administrator kann manuell hinzufügen Benutzer.

Benutzer Liste

Die Liste kann gefiltert auf Benutzernamen (Nutzung * als Platzhalter), Gruppe oder Status. Die Liste kann um von Datum der Eintragung oder Benutzernamen in aufsteigender oder absteigender Reihenfolge.

In diesem Fenster werden Werke der Verwaltung mehrere Benutzer gleichzeitig auf verschiedenen Aktionen :

  • Benutzer löschen (Bestätigung erforderlich)
  • Status ändern
  • assoziierten oder Unternehmen von Gruppen distanzieren
  • Anzeige-Eigenschaften ändern
  • weitere Eigenschaften ändern

Das Ziel ist die SET-Benutzer ausgewählt (Standard) oder alle Benutzer ist gefragt.

Berater

Definition von Benutzer-Berater:

  • den Zugang aller Bildschirme erlaubt durch diesen Status
  • read-only oder nur simulieren-Zugang

Es ist möglich, Berater-Modus mit Variable $conf['allow_adviser'].

Wenn diese Variable ist 'true', ist es möglich zu definieren, wie Benutzer ein Berater
