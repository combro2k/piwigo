Historique

Ces écrans permettent de suivre l'historique de navigation dans PhpWebGallery suivant les options que vous avez déterminées dans - Administration - Générale - Onglet "Historique".

Statistiques

    Si vous avez choisi que les visites des pages index.php et picture.php soient enregistrées dans la table history, cet onglet permet de suivre le nombre de pages visitées

  • Statistiques totales : vous visualisez les statistiques par année
  • Statistiques d'une année : vous visualisez les statistiques des mois de l’année sélectionnée.
  • Statistiques d'un mois : vous visualisez les statistiques des jours du mois sélectionné.
  • Statistiques d'une journée : vous visualisez les statistiques des heures de la journée sélectionnée.

Rechercher

    Si vous avez choisi que les visites des pages index.php et picture.php soient enregistrées dans la table history. Cet onglet permet de suivre plus en détail les pages explorées sur votre site avec l'aide de différents filtres

  • Filtrer par date : sélectionner la plage d'observation dont vous souhaitez observer l’activité
  • Filtrer par type d'élément : sélectionner le type d'élément dont vous souhaitez observer l’activité
  • Utilisateur : sélectionner l'utilisateur dont vous souhaitez observer l’activité
  • Nom du fichier : sélectionner le nom du fichier dont vous souhaitez observer l’activité
  • Identifiant image : sélectionner l'identifiant image dont vous souhaitez observer l’activité
