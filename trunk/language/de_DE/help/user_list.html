Benutzer Liste

Auf dieser Seite können Sie registrierte Benutzer verwalten. Nur Benutzerrechte werden auf verschiedenen anderen Seiten verwaltet.

Hinzufügen eines neuen Nutzers

Der Administrator kann manuell Benutzer hinzufügen.

Benutzer Liste

Die Liste kann gefiltert werden nach Benutzernamen (Eingabe eines * als Platzhalter), Gruppe oder Status. Die Liste kann nach Datum der Registrierung oder Benutzername in aufsteigender oder absteigender Reihenfolge sortiert werden.

In diesem Fenster können bestimmte Eigenschaften von mehreren Benutzern zugleich geändert werden :

  • Benutzer löschen (Bestätigung erforderlich)
  • Status ändern
  • Mitgliedschaft in Gruppen hinzufügen oder entfernen
  • Anzeige-Eigenschaften ändern
  • weitere Eigenschaften ändern

Bei Ziel wird bestimmt, ob die Änderung für markierte Benutzer (Standard) oder alle Benutzer übernommen wird.
