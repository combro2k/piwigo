History

On these screens, you can monitor your gallery visits history according to the options you have chosen in - Administration » Configuration » Options » History index.

Statistics

    If you have chosen to record the(index.php and picture.php) visited pages in the table history, you can monitor them from this tab.

  • Global statistics: visits per year.
  • Yearly statistics: visits per month in the selected year .
  • Monthly statistics: visits per days in the selected month .
  • Daily statistics: visits per hours in the selected day.

Search

    If you have chosen to record the (index.php and picture.php) pages visited in the table history. , you can browse the details of the visited pages from this tab, with the help of different filters.

  • Filter per date : choose the date range you want to observe the activity.
  • Filter per type of element : choose the type of element you want to observe the activity.
  • Filter per user : choose the user you want to observe the activity.
  • Filter per name of the file : choose the file name you want to observe the activity.
  • Filter per image identifier : choose the image identifier you want to observe the activity.
