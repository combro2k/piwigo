Albenberechtigungen verwalten

Diese Seite wird nur bei privaten Alben angezeigt. Hier können Sie definieren, welche Benutzer und Gruppen dieses Album betrachten dürfen.

Gruppen

Sie können Gruppen den Zugriff auf dieses Album verbieten bzw. erlauben, wodurch allen Benutzer der Gruppe der Zugriff auf das Album ermöglicht oder verwehrt wird.

Benutzer

Sie können einzelnen Benutzern den Zugriff auf Alben ermöglichen oder verwehren. Benutzern, denen der Zugriff durch eine Gruppenzugehörigkeit erlaubt wurde, kann mit dieser Option der Zugriff wieder entzogen werden.
