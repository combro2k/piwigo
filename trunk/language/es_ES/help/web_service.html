Web Servicio Checker

Esta página permite definir los parámetros de su web servicio.

Condiciones de funcionamiento
Si el acceso a Web Servicios es controlado (Configuración avanzada => $conf [' ws_access_control ']) usted encuentra aquí todos los valores de la configuración avanzada relativa a Web Servicios.
Llave confidencial (Obligatoria)
Llave única que hay que compartir con su compañero. Debe ser bastante largo (8 carácteres por ejemplo), debe ser bastante complejo (con cifras, mayúsculas y minúsculas, y carácteres especiales). Por ejemplo: "P!e2r!k Le G@2l". Este campo es enmascarado en modo aconsejado.
Blanco (Facultativo)
Sea una lista de identifiants de imágenes esto:
  • list/277,275,142,235,178,190,204,236-238,253,268,276,285,41,73
  • Una lista de categorías como : cat/16,32,21
  • ou encore une liste de tags : tag/22,61,36
Todo pone en una lista de identifiants será reducido tiene su expresión más simple:
  • list/41,73,142,178,190,204,235-238,253,268,275-277,285
Acceso limitado (Facultativo)
Si usted desea limitar a su compañero a un tipo particular de demanda.
Límite de transmision
Número máximo de imágenes enviadas a su compañero a cada una de sus demandas.
Duración
A partir de ahora, indique la disponibilidad en días. Si usted traslada la disponibilidad más allá de su duración, el servicio no debería jamás ser devuelto.
Comentario
Le permite describir quién se encuentra detrás de este Servicio Web, de modo más claro para usted. No olvide que un Administrador en modo aconsejar puede leer el contenido de este comentario.
