Fjern sider

Piwigo gir mulighet for å bruker flere servere å lagre bilder som vil danne ditt galleri. Dette kan være nyttig om ditt galleri er installert på en server med begrenset plass og du har en stor mengde bilder å vise.

  1. rediger file verktøy/create_listing_file_local.inc.php, ved å modifisere seksjons parametrene $conf['prefix_thumbnail'] eller $conf['use_exif'].
  2. ved bruk av ftp, plaser filene verktøy/create_listing_file.php og verktøy/create_listing_file_local.inc.php (modifisert) på din eksterne web side, i den samme mappen som dine album mapper (som mappen gallerier i web siden). For eksempel, la oss si at du kan gå inn på http://example.com/galleries/create_listing_file.php.
  3. gå til administrasjons panelet » Verktøy » Side behandler. Spør om å lage ny side, for eksempel http://example.com/galleries.
  4. en ny ekstern side blir da registrert. Du kan nå gjøre 4 tiltak :
    1. generere en liste : kjør en fjern forespørsel for generere en fjern fil liste
    2. synkroniser : leser den fjerne listing.xml filen og synkroniserer den med database informasjonen
    3. Rens : fjerner fjern listing.xml filer
    4. slett : sletter sidens (og alle relaterte album og bilder) i the databasen
