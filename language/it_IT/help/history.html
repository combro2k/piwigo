Cronoligia

Questi schermi permettono di seguire la cronologia di navigazione del vostro sito a seconda delle optzioni che avete attivatoin - Amministrazione - Generale - Linguetta "Cronologia".

statistiche

    Se avete scelto che le visite delle pagine index.php e picture.php siano registrate nella tabella history, Questa linguetta permette di seguire il numero di pagine visitate

  • Statistiche totali : visualizzate le statistiche annuali
  • Statistiche di un anno : visualizzate le statistiche dei mesi del anno selezionato.
  • Statistiche di un mese : visualizzate le statistiche dei giorni del mese selezionato.
  • Statistiche di una giornata : visualizzate le statistiche delle ore della giornata selezionata.

Ricercare

    Se avete scelto che le visite delle pagine index.php e picture.php siano registrate nella tabella history. Questa linguetta permette di seguire più in dettaglio le pagine visitate sul vostro sito con l'aiuto di diversi filtri.

  • Filtro per data : selezionare l'intervallo d'osservazione di cui volete verificare l’attività.
  • Filtro per tipo di elemento : selezionare il tipo di elemento di cui volete osservare l’attività.
  • Utente : selezionare l'utente di cui volete osservare l'attività.
  • Nome file : selezionare il nome del file di cui volete osservare l'attività.
  • Id immagine : selezionare l'Id dell'immagine che volete osservare l'attività.
