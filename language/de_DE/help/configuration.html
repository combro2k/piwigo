Konfiguration

Dieser Bildschirm ermöglicht die Konfiguration einfach. Denn nur dann, wenn zeigten die Konfigurationseinstellungen, die ausreichen, um die Mehrheit Administratoren. Wenn Sie möchten, die Liste der vielen (und Fortgeschrittene) Einstellungen, lesen Sie die Datei include/config_default.inc.php

Dieser Bildschirm ist in mehrere Abschnitte, in denen die Einstellungen Thema durch.

Hauptsächlich

  • Titel der Galerie: für die RSS-Feeds und E-Mail-Benachrichtigung.
  • Seiten Banner: HTML-Code angezeigt, im oberen Teil der Seiten.
  • URL der Galerie: für die RSS-Feeds.
  • Sperren der Galerie: Sperren der gesamten Galerie für Wartungsarbeiten. Nur Administratoren können den Zugriff auf die Galerie.
  • Rating: Die Bewertung der Fotos ist möglich.
  • Rating von den Besuchern: Auch nicht registrierte Benutzer können Bilder Satz.
  • Lassen Sie Benutzer-Registrierung: Die Registrierung ist kostenlos für alle.
  • E-Mail-Adresse ist obligatorisch für alle Nutzer: E-Mail-Adresse wird überprüft bei der Registrierung oder Profil aktualisieren, außer wenn die Maßnahme erfolgt auf Verwaltung Teil.
  • E-Mail-Administratoren, wenn ein neuer Benutzer registriert: Administratoren werden erhalten für jede Registrierung.

Historisches

Besuche auf den Seiten index.php und picture.php werden in der History Tabelle.

Besuche werden in Administration, Specials, Geschichte.

  • Seite speichern Besuche von Gästen: Seite Besuche von Gästen werden gespeichert.
  • Seite speichern Besuche von Nutzern: Besuche von registrered Benutzer gespeichert werden.
  • Seite speichern Besuche von Administratoren: Seite Besuche von Administratoren gespeichert werden.

Kommentare

  • Kommentare für alle: Auch nicht registrierte Gäste können nach Bemerkungen.
  • Anzahl der Kommentare pro Seite.
  • Validierung: Ein Administrator muss die Validierung Benutzer Kommentare bevor sie sichtbar in der öffentlichen Teil. Die Validierung der Benutzer Kommentare erfolgt in dem Bildschirm Verwaltung, Bilder, Kommentare.
  • Administratoren benachrichtigen, wenn ein Kommentar eingetragen is: Sende eine E-Mail an den Administrator, wenn ein Benutzer registriert ist, die einen Kommentar validiert.
  • Administratoren benachrichtigen, wenn ein Kommentar der Gültigkeitsprüfung erforderlich : Sende eine E-Mail an den Administrator, wenn ein Benutzer einen Kommentar registriert, die eine Validierung der Administratoren. Die Validierung der Benutzer Bewertungen erfolgt auf dem Bildschirm Verwaltung, Bilder, Kommentare.

Upload

  • Show-Upload-Link jedes Mal: Wenn vorhanden uploadeable Kategorien, fügen Link wird zeigen, für jede Kategorie.
  • User-Ebene Zugang zum Hochladen: Ermöglicht zu beschränken, Upload von den Nutzern
  • E-Mail-Verwaltung, wenn ein Bild hochgeladen: Administratoren werden erhalten für jede E-Mail Bild hochgeladen von einem Benutzer.

Standardansicht

Ändern der Standardansicht: für die Besucher nicht verbunden. Nachdem Sie sich angemeldet haben, diese Optionen sind überlastet durch die von Benutzer, es kann auf dem Bildschirm Profil.

Es ist möglich, diese Optionen, um die vorhandenen Benutzer, aber es handelt sich um einen anderen Bildschirm: Verwaltung, Identifikation, Benutzer, in dem Sie können diese Optionen für eine Liste der ausgewählten Benutzer.

  • Sprache: nur die Labels Piwigo. Die Namen der Kategorien, Bild und Beschreibung sind nicht lokalisiert (zB mehrsprachig).
  • Anzahl der Bilder pro Zeile
  • Anzahl der Zeilen pro Seite
  • Interface Thema
  • Jüngster Zeit: Mit dem Tag. Frist, innerhalb eines Bildes ist gezeigt, wie neue. Müssen überlegen 1 Tag.
  • Erweitern Sie alle Kategorien: Erweitern Sie alle Kategorien von standardmäßig in das Menü? Warnung: diese Option ist Ressourcen verbrauchen und Mai eine große angezeigt, sofern Sie eine Kategorie Baum ist enthält viele Kategorien.
  • Zeige die Anzahl der Kommentare: Anzeige der Anzahl der Kommentare für jedes Bild auf die Vorschaubilder Seite. Resource sein.
  • Anzeige der Treffer zählen: Display beobachtet Treffer auf das Bild nur im Rahmen seiner Thumbnail auf die Vorschaubilder Seite. Nur wenn Advanced Configuration Parameter ist:
    $conf [ 'show_nb_hits'] = true;
    Hinweis: Es ist false per Default.
  • Maximale Breite der Bilder: maximal angezeigt Breite. Wenn Bilder sind größer als diese Einstellung, werden sie auf die Größe Anzeige. Wenn man bedenkt, Füllung diese Option, die Sie besser prüfen Änderung der Breite der Sie Ihre Bilder statt.
  • Maximale Höhe der Bilder: gleichen Bemerkungen wie für vorherige Einstellung.
