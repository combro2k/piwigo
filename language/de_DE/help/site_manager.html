Remote-Sites

Piwigo bietet die Möglichkeit, mehrere Websites für Speichern der Dateien, aus denen Ihre Galerie. Dies kann nützlich sein, wenn Ihre Galerie befindet sich auf einem Speicherplatz begrenzt ist und Sie haben viele Dateien zu zeigen.

  1. Sie die Datei tools / create_listing_file.php durch eine Änderung der Abschnitt Parameter wie $ conf [ 'prefix_thumbnail'] oder $ conf [ 'use_exif'].
  2. die Datei tools / create_listing_file.php geändert auf Ihrer Website Remote, im selben Verzeichnis wie die Vertreter Ihrer Verzeichnisse Kategorien (wie das Verzeichnis Galerien von dieser Seite) per FTP. Zum Beispiel, dass Sie Zugriff auf http://exemple.com/galleries/create_listing_file.php.
  3. Bewußt auf Verwaltung, Kategorien, Betreiber der Seite. Verlangen eine neue Website, zB http://exemple.com/galleries.

  4. eine neue Remote-Site gespeichert wird. 4 Aktionen möglich:

    1. die Liste: startet eine Remote-Abfrage für Generieren Sie die Remote-Liste
    2. synchronisieren: Gibt die Remote-Datei listing.xml und synchronisiert mit der Datenbank Ort
    3. reinigen: Der Remote-Datei von List
    4. zerstören: Der Standort (und alle Elemente die damit verbundenen) in der Datenbank
