Sites distants

Piwigo offre la possibilité d'utiliser plusieurs sites pour stocker les fichiers qui composeront votre galerie. Cela peut être utile si votre galerie est installée sur un espace de stockage limité et que vous ayez de nombreux fichiers à montrer.

  1. éditer le fichier tools/create_listing_file_local.inc.php en modifiant la section des paramètres comme $conf['prefix_thumbnail'] ou $conf['use_exif'].
  2. par FTP, placer les fichiers tools/create_listing_file.php et tools/create_listing_file_local.inc.php (modifié) sur votre site distant, dans le même répertoire que les répertoires représentant vos albums (comme le répertoire galleries de ce site). Par exemple, disons que vous pouvez accéder à http://exemple.com/galleries/create_listing_file.php.
  3. se rendre sur Administration » Outils » Gestionnaire des sites. Demander à créer un nouveau site, par exemple http://exemple.com/galleries.

  4. un nouveau site distant est enregistré. 4 actions sont possibles :

    1. générer la liste : lance une requête distante pour générer le fichier de listing distant
    2. synchroniser : lit le fichier distant listing.xml et synchronise avec la base de données locale
    3. nettoyer : supprime le fichier distant de listing
    4. détruire : supprime le site (et toutes les informations qui y sont associées) dans la base de données
