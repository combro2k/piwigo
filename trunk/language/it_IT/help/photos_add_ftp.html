Guida rapida
  1. Create una directory sul vostro PC.
  2. Copiate qualche foto in questa directory, e ridimensionatele per la visualizzazione sul web. Attenzione: il nome delle directory e dei files deve essere solamente formato da lettere, cifre e/o i simboli "-", "_" o ".". Nessuno spazio o caratteri accentati.
  3. Con un client FTP, caricate la directory nella directory "galleries" della vostra gallery Piwigo .
  4. Connettetevi alla vostra gallery Piwigo e andate in Amministrazione » Immagini » Miniature per creare le miniature mancanti.
  5. Andate in Amministrazione e cliccare sul grosso pulsante Sincronizzazione.

Bene! Avete appena creato la prima categoria della vostra gallery.
