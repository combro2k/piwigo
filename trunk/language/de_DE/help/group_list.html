Gruppenliste und -verwaltung

Auf dieser Seite können Sie Gruppen verwalten.

Hinzufügen einer Gruppe

Der Administrator kann Gruppen mit frei wählbarem Namen hinzufügen.

Liste der Gruppen

In der Liste kann für jede Gruppe die Mitgliederliste anschauen. Es wird die Seite der Benutzerverwaltung mit einem automatischen Filter angezeigt.

Darüberhinaus können drei Aktionen für jede Gruppe gewählt werden.

  • Die Zugriffsrechte zu privaten Alben können verändert werden.
  • Eine Gruppe kann gelöscht werden (eine Bestätigung wird gefordert).
  • Eine Gruppe kann als Standardgruppe ausgewählt werden (eine Bestätigung wird gefordert).
Standard-Gruppen
Neue Nutzer werden automatisch zur Standardgruppe hinzugefügt.
