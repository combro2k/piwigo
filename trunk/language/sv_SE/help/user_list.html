Lista på användare

listan kan filtreras på användarnamn (använd * som jokertecken), en grupp eller status. Den kan sorteras genom registrering datum eller användarnamn, i stigande eller fallande ordning.

I denna meny kan du hantera flera användare samtidigt, med hjälp av olika åtgärder:

  • ta bort användare (kräver bekräftelse)
  • ändra status
  • koppla eller separera från grupper
  • ändra visningsegenskaper
  • ändra ytterligare egenskape

Som standard är målet användaren eller alla användare som visas i den filtrerade listan.

Rådgivare

Definition av en rådgivare användare: som har läst/simulering på alla menyer som tillåts av sina egna behörigheter.

rådgivare läge är inställt med variabeln $conf['allow_adviser'].

Om denna variabel är satt till "sant", kan alla användare vara rådgivare
