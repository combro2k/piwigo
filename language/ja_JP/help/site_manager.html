リモートサイト

Piwigoでは、イメージを保存して、あなたのギャラリーを構成するため、 複数のサーバを使用することができます。 これは、ギャラリーがインストールされているサーバのディスクスペースが限られていて、 あなたが大量のイメージを表示したい場合、有用です。

  1. $conf['prefix_thumbnail'] または $conf['use_exif']
    2. のようなパラメータセクションを変更することで、 ファイルtools/create_listing_file.phpを編集してください。
  2. 修正済みファイルtools/create_listing_file.phpを あなたのカテゴリのディレクトリ (このウェブサイトのディレクトリgalleries )と 同じ遠隔サーバのディレクトリにFTPでアップロードしてください。 例えば、あなたがhttp://example.com/galleries/create_listing_file.phpにアクセスできるようにします。
  3. 「管理パネル > カテゴリ > サイトマネージャ」に移動してください。 例えば、http://example.com/galleriesのようなサイトを作成してください。
  4. 新しいリモートサイトが作成されます。あなたは、次の4つの処理を実行できます :
    1. 一覧の作成 : 遠隔ファイル一覧を作成するリクエストを送信できます。
    2. 同期化 : 遠隔サイトのlisting.xmlファイルを読んで、 データベース情報と同期化できます。
    3. クリーン : 遠隔サイトのlisting.xmlファイルを削除できます。
    4. 削除 : データベース内のサイト (および、すべての関連カテゴリおよびエレメント) をデータベースから削除できます。
