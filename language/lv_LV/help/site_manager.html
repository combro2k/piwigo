Attālinātās web lapas

Piwigo piedāvā iespēju Jūsu galerijas veidošanai attēlu glabāšanai izmantot vairākus serverus. Tas ir noderīgi, ja Jūsu galerija ir uzinstalēta uz mazietilpīga servera un Jums ir jāparāda liels daudzums attēlu.

  1. rediģē failu tools/create_listing_file_local.inc.php, modificējot parametru daļu kā $conf['prefix_thumbnail'] vai $conf['use_exif'].
  2. lietojot ftp, izvieto failus tools/create_listing_file.php un tools/create_listing_file_local.inc.php (modificēti) uz Jūsu attālinātās web lapas, tajā pašā mapē, kur atrodas Jūsu kategoriju mapes (kā mape galerijas uz šīs web laps). Piemēram, teiksim Jums ir piekļuve http://example.com/galleries/create_listing_file.php.
  3. ejat uz administrēšanas paneli, Kategorijas, Web lapas pārvaldnieks. Izvēlaties izveidot jaunu web lapu, piemēram http://example.com/galleries.
  4. tiek reģistrēta jauna attālināta web lapa. Ispējams veikt 4 darbības :
    1. sarakstu ģenerēšanu : startē attālinātu pieprasījumu ģenerēt attālināto failu sarakstu
    2. sinhronizāciju : nolasa attālināto listing.xml failu un sinhronizē ar informāciju datu bāzē
    3. tīrīšanu : dzēš attālināto listing.xml failu
    4. dzēšana : dzēš web lapu (un visas saistītās kategorijas un elementus) datu bāzē
