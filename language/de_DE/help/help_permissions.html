Berechtigungen für Fotos

Jedem Foto ist eine Berechtigungsstufe (eine Sperre) zugewiesen, genauso wie jeder Benutzer eine gewisse Berechtigungsstufe besitzt (ein Schlüssel). Es gibt fünf Berechtigungsstufen:

  1. Keine Berechtigung
  2. Kontakte
  3. Freunde (eine Stufe höher als Kontakte)
  4. Familie (eine Stufe höher als Freunde)
  5. Administratoren (die höchste Berechtigungsstufe)

Eine höhere Stufe erbt alle Berechtigungen der niedrigeren Stufen. Für einen Benutzer gilt: Je höher seine Berechtigungsstufe, desto mehr Fotos kann er sehn.

Ein Beispiel: Wenn dem Foto "peter_hochzeit-0024.jpg" die Berechtigungsstufe "Familie" zugewiesen ist, dann:

  • kann Peter (der ein "Administrator" ist) das Foto sehn, weil "Administratoren" alle Fotos sehen können
  • kann Elisabeth (die Teil der "Familie" ist) das Foto ebenfalls sehen
  • kann Marie (die nur ein "Freund" ist) das Foto nicht sehen

Ein Benutzer der keine Berechtigungen hat, den Inhalt einer Kategorie zu sehen, kann weder die Kategorie selbst, noch den Titel der Kategorie sehn. Das selbe Prinzip wird auch auf Bezeichnungen angewendet.

Sie können die Berechtigungsstufe eines Benutzers unter Administration » Benutzer » Verwalten sehn.

Berechtigungen für Kategorien

Wenn die definierten Berechtigungsstufen nicht Ihren Bedürfnissen entsprechen, haben Sie auch die Möglichkeit, Benutzern oder Gruppen Berechtigungen auf Kategorien zu vergeben. Sie können problemlos Berechtigungen für Kategorien und Fotos parallel verwenden.

Um Berechtigungen einer Kategorie zu verwalten, müssen Sie die Zugangsart der Kategorie auf "privat" setzen.

Sie können eine Kategorie auf "privat" setzen, indem Sie eine einzelne Kategorie bearbeiten (Administration » Kategorien » Verwalten » Bearbeiten) oder indem Sie Eigenschaften für alle Kategorien festlegen (Administration » Kategorien » Eigenschaften » Öffentlich / Privat).

Wenn eine Kategorie als "privat" markiert ist, können Sie die Berechtigungen für Benutzer und Gruppen an drei unterschiedlichen Stellen vornehmen:

  • Administration » Benutzer » Verwalten » Aktionen, Berechtigungen (jeweils ein Link pro Benutzer)
  • Administration » Benutzer » Gruppen » Aktionen, Berechtigungen (jeweils ein Link pro Gruppe)
  • Administration » Kategorien » Verwalten » Aktionen, Berechtigungen für Kategorie ändern (jeweils ein Link pro Kategorie)
