Berechtigungen für Fotos

Jedem Foto ist eine Berechtigungsstufe (eine Sperre) zugewiesen, genauso wie jeder Benutzer eine gewisse Berechtigungsstufe besitzt (ein Schlüssel). Es gibt fünf Berechtigungsstufen:

  1. Keine Berechtigung
  2. Kontakte
  3. Freunde (eine Stufe höher als Kontakte)
  4. Familie (eine Stufe höher als Freunde)
  5. Administratoren (die höchste Berechtigungsstufe)

Eine höhere Stufe erbt alle Berechtigungen der niedrigeren Stufen. Für einen Benutzer gilt: Je höher seine Berechtigungsstufe, desto mehr Fotos kann er sehen.

Ein Beispiel: Wenn dem Foto "peter_hochzeit-0024.jpg" die Berechtigungsstufe "Familie" zugewiesen ist, dann:

  • kann Peter (der ein "Administrator" ist) das Foto sehen, weil "Administratoren" alle Fotos sehen können
  • kann Elisabeth (die Teil der "Familie" ist) das Foto ebenfalls sehen
  • kann Marie (die nur ein "Freund" ist) das Foto nicht sehen

Ein Benutzer, der keine Berechtigungen hat den Inhalt eines Albums zu sehen, kann weder das Album selbst, noch den Titel des Albums sehn. Das selbe Prinzip wird auch auf "Tags" angewandt.

Sie können die Berechtigungsstufe eines Benutzers unter Administration » Benutzer » Verwalten sehen.

Berechtigungen für Alben

Wenn die definierten Berechtigungsstufen nicht Ihren Bedürfnissen entsprechen, haben Sie auch die Möglichkeit, Benutzern oder Gruppen Berechtigungen auf Alben zu vergeben. Sie können problemlos Berechtigungen für Alben und Fotos parallel verwenden.

Um Berechtigungen eines Albums zu verwalten, müssen Sie die Zugangsart des Albums auf "privat" setzen.

Sie können ein Album auf "privat" setzen, indem Sie ein einzelnes Album bearbeiten (Administration » Alben » Verwalten » Bearbeiten) oder indem Sie Eigenschaften für alle Alben festlegen (Administration » Alben » Eigenschaften » Öffentlich / Privat).

Wenn ein Album als "privat" markiert ist, können Sie die Berechtigungen für Benutzer und Gruppen an drei unterschiedlichen Stellen vornehmen:

  • Administration » Benutzer » Verwalten » Aktionen, Berechtigungen (jeweils ein Link pro Benutzer)
  • Administration » Benutzer » Gruppen » Aktionen, Berechtigungen (jeweils ein Link pro Gruppe)
  • Administration » Alben » Verwalten » Aktionen, Berechtigungen für Album ändern (jeweils ein Link pro Album)
