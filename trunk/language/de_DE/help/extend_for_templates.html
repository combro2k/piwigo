Konfiguration zu "erweitern" die Templates

Ziel

Festlegung der Bedingungen, die zur Ersetzung des Templates (Vorlage) ab Template-Erweiterung (Template "replacer" (Stellvertreter)).

"Templates Ursprungs" aus./template/yoga (oder einem anderen Ordner wie "yoga") durch einen Stellvertreter, wenn dieser im Zusammenhang mit seiner ursprünglichen Vorlage (und auf freiwilliger Basis, wenn die Adresse (URL) beantragt enthält eine URL contains eine optionale Parameter in der URL ").

Die "optionale Parameter in der URL" sind die das sie finden nach dem Namen des Moduls in die Adresse (URL).

So ein "Optionale Parameter in der URL" kann ein "permalink" aktiv (siehe permalinks in unserer Dokumentation für weitere Informationen).

Lesen Sie die `Warnungen` zu den Nutzungsbedingungen.

Was sind Template-Erweiterungen?

Diese Konfiguration ist ein leistungsfähiges Werkzeug für Einsteiger und fortgeschrittene Anwender. Mit dem FTP, in der Vorlage-Erweiterung Ordner, der Webmaster kann Duplikat eine Menge von der verteilt. tpl-Dateien ( "Original-Vorlage") mit einem anderen Namen ( "Replacer"). "Original-Vorlagen", aktuelle Original-Vorlagen unterstützt werden:

  • index.tpl
  • mainpage_categories.tpl and/or thumbnails.tpl
  • menubar.tpl
  • header.tpl and/or footer.tpl
  • picture_content.tpl and/or picture.tpl
  • profile.tpl and/or profile_content.tpl
  • tags.tpl
  • upload.tpl
  • about.tpl
  • popuphelp.tpl
  • search.tpl and/or search_rules.tpl
  • nbm.tpl and/or notification.tpl
  • identification.tpl and/or register.tpl
  • redirect.tpl

Wo finde ich "Original-Vorlagen"?

Verteilte Original-Vorlagen sind in template /yoga/ (nicht bearbeiten sie im Ort für Upgrade Perspektiven). Also mit "Original-Vorlagen", jeder Webmaster kann seine "replacers (benutzerdefinierte Vorlagen)".

Wo kann ich sparen "replacers"?

Replacers (benutzerdefinierte Vorlagen) sind in der Template-Erweiterung Ordner (oder in seinen Unterordnern). Replacers kann eine beliebige Datei mit der Erweiterung. Tpl-Erweiterung. Zum Beispiel: template-extension/my-extension/video.tpl.

Wie aktiviere ich meine "Ersatz"?

Ein Original-Vorlage sollte für jede Replacer (benutzerdefinierte Vorlagen), um die damit verbundenen Ersatz. Dies ist das Ziel der "Extend für Vorlagen" in der Konfiguration Administration Menü.

Wenn Sie ein optional URL Keyword, dann Ersatz werden jeweils nur auf Seiten mit diesem Keyword. Zum Beispiel: index.php? / most_visited

Advanced-User nutzen werden Dauerlinks als optional URL Keyword.

If you select a Bound template then replacements will be operate only on this template.

Wie kann ich deaktivieren meine "replacers"?

  • Ausschluss der "Template Herkunft".
  • Per FTP löschen "ersetzt".

Warnhinweise

  • Original-Vorlage muss das "Original"-Vorlage ansonsten können unvorhersehbare Ergebnisse!
  • Auf mehreren replacers (benutzerdefinierte Vorlagen) mit dem gleichen Original-Vorlage und die gleichen optional URL Keyword nur die letzte Replacer aktiv ist.
  • Nicht selektierte optional URL Keyword wird die aktive Replacer auf jeder Vorlage Nutzung. Wie vorherigen Punkt, um können vorherigen ausgewählten replacers.
  • Unterordner: Unterordner werden ignoriert, aber sie sind nützlich, eine Änderung der derzeitigen Reihenfolge.
    distributed-Proben: für Test-Zwecke verteilt werden! nicht bearbeiten Erstellen Sie Ihren eigenen sind.
  • Am löschen eines Replacer der Original-Vorlage ist sofort reaktiviert.
