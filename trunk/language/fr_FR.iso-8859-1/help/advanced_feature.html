Fonctions avancées

Liste des fonctionnalités spéciales pour les administrateurs experts de PWG.

  • Eléments non liés : Affiche les images qui ne sont pas associées à une catégorie virtuelle
  • Fichiers portant le même nom dans plusieurs catégories physiques : Affiche les images ou les éventuels fichiers qui pourraient être trouvés dans plusieurs catégories.
