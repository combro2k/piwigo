Lage Miniatyr bilder

Denne siden lar deg lage manglende miniatyr bilder. Det er ikke anbefalt å bruker denne funksjonen dersom ditt galleri ligger på en delt server.

GD versjon
GD er bilde manipulerings bibliotek for PHP. Velg den versjon som er installert på din server. Du vil bli advart med en feil melding , om du skulle velge feil; bare gå tilbake til forrige side med nettleseren og velg en annen versjon. Om ikke versjon virker, støtter ikke din server GD.
Bredde og høyde
Maksimum miniatyr bilde dimensjoner. Orginal ratio er satt som standard. Standard verdier kan endres ved bruk av konfigurasjons instillingene tn_width og tn_height (se include/config_default.inc.php fil).
Antall miniatyr bilder som skal lages
Prøv ikke å lage for mange miatyrer av bilder samtidig. Denne prosessen bruker mye CPU kraft. Om ditt Piwigo gallery er hosted av en gratis tjeneste, for høy CPU bruk kan føre til begrensninger på web siden din.
Fil format
Miniatyr bilder blir laget i JPEG fil format. De kan kun bli laget fra PNG og JPEG bilder.
