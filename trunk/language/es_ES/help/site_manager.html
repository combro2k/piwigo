Sitios distantes

Piwigo ofrece la posibilidad de utilizar varios sitios para almacenar los ficheros que compondrán su galería. Esto puede ser útil si su galería está instalada sobre un espacio de almacenamiento limitado y si usted tiene numerosos ficheros que hay que mostrar.

  1. editar el fichero tools/create_listing_file.php modificando la sección de los parámetros como $conf['prefix_thumbnail'] or $conf['use_exif'].
  2. colocar el fichero tools/create_listing_file.php modificado sobre su sitio distante, en el mismo repertorio que los repertorios que representan sus categorías (como el repertorio galleries de ce site) para FTP. Por ejemplo, digamos que usted puede acceder a http://exemple.com/galleries/create_listing_file.php.
  3. Ir aAdministración, Categorías, Gestion de las sitios. Pedir por crear un nuevo sitio, por ejemplo http://exemple.com/galleries.

  4. Un nuevo sitio distante esta registrado. 4 acciones posibles :

    1. generar la lista : lanza una demanda distante para generar el fichero de listado distante
    2. sincronizar : lee el fichero distante listing.xml y sincroniza con la base de datos local
    3. limpiar : Suprime el fichero distante de listado
    4. destruir : suprime el sitio (y todos los elementos que estan asociados con el) en la base de datos
