Bearbeiten einer Kategorie

Informationen

  • Namen: Umbenennen der Kategorie (praktisch für Kategorie Physik).
  • Beschreibung: eine Vorstellung vom Inhalt der Kategorie, Diese Angabe wird unter der Seite der Miniaturen.

verschieben

Wenn die virtuelle Klasse ist, können Sie diese zu verschieben. Verschieben eines Kategorie bedeutet, ändern Kategorie angeben.

Optionen

  • Art des Zugriffs: Verwaltung der Zugriffsrechte. Wenn du eine private, alle Mädchen werden private Gruppen. Wenn Sie besuchen eine öffentliche, allen Kategorien Mütter werden werden.
  • sperren: die Kategorie und alle Kategorien Mädchen werden für Wartungsarbeiten vorübergehend nicht erreichbar.
  • Kommentare: die Benutzer kommentieren die Elemente dieser Kategorie.
  • zulassen Hinzufügen von Bildern: die Benutzer zum Herunterladen von Dateien. Diese Option ist nur verfügbar, wenn die Kategorie Physik (und nicht-virtuelle).

Sie können diese Optionen auch die Verwaltung durch die Verwaltung der "Eigenschaften" (Bildschirme hinzufügen, Kommentare, "Schützen", Public / Private, Vertreter ab d 'Verwaltung, Kategorien, Eigenschaften).

Sortieren

Verwenden Sie die Sortierung der Bilder Standard.

Angabe, ob die Sortierung gilt auch für die Unterkategorien.

Wählen Sie die Sortierung der Spalten zu verwenden, um diese Kategorie.

  • Erstellungsdatum: Datum der Aufnahme
  • Datum:: Datum der Synchronisierung
  • Durchschnitt (*): der Durchschnitt werden durch die Besucher
  • Weitere Ansichten (*): Der Besuch der Kurse können die Reihenfolge der Ansichten
  • Dateiname: Name der Datei in den Informationen oben
  • Id: internen Id (die jüngsten sind "Id" größer als die früheren).

(*) Achtung: Testen Sie selbst, die diese Sortierungen können unvorhersehbare Ergebnisse.

Vertreter

Der Vertreter einer Kategorie ist die Miniaturansicht angezeigt Vertretung der Kategorie auf der Hauptseite (category.php), wenn eine Kategorie nur Unterkategorien und keine direkten Elemente (wie in der Stamm)

Es gibt 4 Möglichkeiten zur Auswahl der Vertreter einer Kategorie:

  • Seite beim Anzeigen von Bildern: in der Schaltflächen Maßnahmen, die einer von ihnen können Sie das Element angezeigt als Vertreter der Kategorie angezeigt. Diese Schaltfläche ist nur für die Administratoren.
  • Bearbeiten von Informationen zu einem Bild in der Verwaltung. Dieser Bildschirm kann von picture.php oder Verwaltung von Los im Modus. Finden Sie in der Hilfe auf diesem Bildschirm für die Details.
  • Verwaltung, Kategorien, Eigenschaften, Vertreter. Finden Sie in der Hilfe für diesen Bildschirm, um die Details.
  • Bearbeiten einer Kategorie (in diesem Bildschirm).

Die Darstellung hängt von der Einstellung der Konfiguration allow_random_representative (siehe include / config_default.inc.php).

Standard (allow_random_representative auf false) jeder Kategorie, die mindestens ein Element wird durch ein Element fest. Einmal festgelegt (die Einführung der Kategorie), in dem der Vertreter ändert nur, wenn der Administrator den Antrag. Wenn der Vertreter nicht ist nicht, Sie können zu fragen Ein neuer Vertreter Zufall.

Wenn die Einstellung allow_random_representative wahr ist, eine Kategorie enthalten, kann nicht von Vertreter bestimmt. Verwenden Sie einfach auf die Schaltfläche löschen Vertreter.

Wenn die Kategorie enthält keine Anhaltspunkte (aber nur Unterkategorien) sie kann sich vertreten durch irgendeine Element auf dem Bildschirm durch Informationen bearbeiten Ein Bild. Die einzige Möglichkeit, dieses Fenster ist die Schaltfläche Vertreter löschen.
