Siti in remoto

Piwigo offre la possibilità di usare diversi server per immagazzinare le immagini della vostra galleria. Ciò può essere utile se la vostra galleria è installata su uno spazio limitato e avete una grande quantità d'immagini da mostrare.

  1. modificare il file tools/create_listing_file.php agendo sui parametri come $conf['prefix_thumbnail'] o $conf['use_exif'].
  2. piazzare il file tools/create_listing_file.php modificato sul Vostro sito remoto, nella stessa directory che le categorie che compongono la vostra galleria (come la directory galleries di questo sito), per FTP. Per esempio, supponiamo che potete accedere a http://esempio.it/galleries/create_listing_file.php.
  3. andare su Amministrazione, Categorie, Gestione dei siti. Richiedere la creazione di un nuovo sito, per esempio http://esempio.it/galleries.

  4. Un nuovo sito remoto è registrato. 4 azioni sono ora possibili :

    1. generare la lista : lancia una richiesta remota per generare il file di listing remoto
    2. sincronizzare : legge il file remoto listing.xml e sincronizza con la base dati locale.
    3. pulire : cancella il file di listing remoto.
    4. Cancellare : cancella il sito remoto (e tutte le relative categorie ed elementi associati) dalla base dati.
